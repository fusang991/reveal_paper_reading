# Paper Reading Slides

这是一个最小可用的 `Markdown -> reveal.js` 演示文档项目骨架。

## 目录

- `notes/`：原始阅读笔记
- `slides/`：用于演示的 Markdown
- `docs/`：项目说明与 reveal.js 使用文档
- `src/`：演示页面入口和样式

## 快速开始

```bash
npm install
npm run dev
```

然后打开：

```text
http://localhost:8000/src/
```

默认会加载 `slides/demo.md`。
