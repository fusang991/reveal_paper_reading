# Markdown 基础 Demo

这是一个更简单的 Markdown 演示稿，只保留最常见、最基础的写法。

---

## 标题和正文

Markdown 用 `#` 表示标题：

# 一级标题

## 二级标题

### 三级标题

普通文字直接写就可以。

---

## 无序列表

- 阅读论文
- 提取重点
- 整理笔记
- 准备汇报

---

## 有序列表

1. 先看摘要和结论
2. 再看方法和实验
3. 最后整理自己的理解

---

## 强调和行内代码

可以使用 **加粗** 表示重点。

可以使用 *斜体* 表示补充说明。

可以使用 `notes/`、`slides/` 这样的行内代码表示文件名或命令。

---

## 引用

> 好的笔记不是把论文全部复制下来，而是留下自己真正理解的部分。

引用常用于记录原文观点、重要结论或自己的总结。

---

## 代码块

```python
from pathlib import Path

text = Path("notes/paper.md").read_text(encoding="utf-8")
print(text[:100])
```

代码块适合放脚本、命令或伪代码。

## 一页一个重点

写 slide 时可以尽量做到：

- 每页只讲一个主题
- 列表不要太长
- 代码只放关键部分
- 结论写得直接一点
